This sections only detail GUI part

Requested features for user:

  • Fix multi target support bugs:
    • plot image and plot uv map always return first target results
    • associated setting.i mixes up parameters of different targets
  • Add close button
  • Display a progress bar during long actions( user sometimes click several times on buttons because nothing seems running...)
  • Enhance error message displayed that occures on yorick side
  • Propose on parameters view that group every parameter of one composed model
  • Request to the server the synthetic oifits file that according given model
  • Finish result plots
    • radial plots (data selection should be interactive)
    • image (fix ticks legend...)
    • sniffer plot
  • Use checkboxes to file selection instead of the simple list widget
  • Permit user to select installed programs as fits viewer
  • Add document modified flag into the whole code . That is really important because work can be lost after load or new menu request!
  • Add one format version of all xml settings files (Future revision will be able to check format validity and propose converter tools)
